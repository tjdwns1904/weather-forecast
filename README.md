## 📝 소개
기상예보 사이트 입니다.    
   
검색창을 통해 도시 이름을 검색하여 해당 도시의 기상정보를 불러올 수 있습니다.   
사용자의 위치에 기반하여 현재 위치하고 있는 도시의 기상정보도 불러올 수 있습니다.   
그 뿐만 아니라, 자주 검색하는 도시들을 즐겨찾기하여 검색할 필요없이 기상정보를 불러올 수 있습니다.
